# Plan de Bornage - Parcelle 159 13 03

Reproduction HTML/CSS d'un plan de bornage officiel du Burkina Faso.

## Description

Ce projet contient la reproduction numérique d'un plan de bornage cadastral pour une parcelle située à Daguilma, commune de Loumbila, province d'Oubritenga.

## Informations de la parcelle

- **Parcelle**: 159 13 03
- **Superficie**: 245 m²
- **Village**: DAGUILMA
- **Commune**: LOUMBILA
- **Province**: OUBRITENGA
- **Centre**: ZINIARE
- **Cedant**: Etat Burkinabé
- **Cessionnaire**: GROUP KASTOR AFRICA

## Fichiers du projet

- `plan_de_bornage.html` - Fichier HTML principal contenant la structure du document

- `plan de bornage.pdf` - Document PDF original

## Utilisation

Pour visualiser le plan de bornage :

1. Ouvrez le fichier `plan_de_bornage.html` dans votre navigateur web
2. Le fichier CSS sera automatiquement chargé
3. Le document s'affichera avec la mise en forme appropriée

```bash
# Ouvrir avec le navigateur par défaut (Linux)
xdg-open plan_de_bornage.html

# Ou simplement double-cliquer sur le fichier
```

- Le document utilise des unités relatives pour une meilleure adaptabilité
- Les hachures de la parcelle sont créées avec `repeating-linear-gradient`
- La boussole et les éléments de positionnement utilisent `position: absolute`
- Le design est optimisé pour l'impression et l'affichage écran
